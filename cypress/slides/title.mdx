## Example

```
// visits URL /campaigns/create
cy.visit('/campaigns/create');

// type text in input field
cy.get('[data-testid="campaignName"]').type("Campaign title test");

// clicks load button
cy.get('[data-testid="loadButton"]').click();
```

<!-- note
Example for browser automation
cy.get: targets element
clicks on an element
-->

## Stubbing network call example

```
cy.server(); // tell cypress to start listening to network call

// load mock response json file
cy.fixture('getContractId.json').then((response) => {
    // listen to getContractId API call and stub response
    cy.route({
        method: "POST",
        url: '/api/contract/getContractId',
        response
    }).as("getContractId");
});
```

<!-- note
Example for stubbing API call
Whenever we call API, the response will be the content of getContractId.json
-->

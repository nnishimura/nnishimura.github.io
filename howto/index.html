    （前回より）動きをなめらかにする

    CSS

                .anime{
                /*transition: CSSのプロパティ アニメーションの期間秒数 イージングの種類 遅れ秒数;*/
                transition: background-color 1s linear 0s;
                transition: background-color 1s cubic-bezier(0.250, 0.250, 0.750, 0.750) 0s;
                 /* カンマで区切ることで、複数指定ができる */
                transition:background-color 0.5s linear 0s, width 0.5s linear 0s;
                 /* allですべてのプロパティを指定 */
                transition: all 1s cubic-bezier(0.250, 0.250, 0.750, 0.750) 0s;
            }

  • transform

    「変形=transform」する

    CSS

    .translate3d{
  /*  transform: translate3d(X,Y,Z); */
  /*  X方向に50px,Y方向に-20pxずれる */
  transform: translate3d(50px,-20px,0);
}
.scale{
  /*  transform: scale(X,Y); */
  /*  X方向に1.2倍、Y方向に1.2倍拡大する */
  transform: scale(1.2,1.2);
}
.rotate{
  /*  transform: rotate(角度); */
  /*  時計回りに360度回転する。 */
  transform: rotate(360deg);
}
.skew{
  /*  transform: skew(X軸の傾斜角度,Y軸の傾斜角度); */
  /*  X軸にそって20度変形する */
  transform: skew(20deg,0deg);
}

    イージングの種類はこちら

    translate3d

    scale

    rotate

    skew

  • animation

    繰り返しのアニメーション。
    繰り返し再生したいアニメーションがあるときはこれを使う。
    自分で一から作るのではなく、サンプルを改造できるくらいの知識で良い。

    CSS

    /* animation: アニメーション名 秒数 イージング 繰り返し回数; */
.anime{
  animation: buttonFlash 1s linear infinite;
  /*ほとんどの場合infinite。その他はtransitionで事足りる*/
}
@keyframes buttonFlash {
  /* 経過時間の割合 */
  0% {
  /* 経過時間が0%のとき、opacity:0.2 */
    opacity: 0.2;
  }
  8% {
    opacity: 1;
  }
  100% {
    opacity: 0.2;
  }
}
